# Cash Control

## Overview 

Cash Control is a personal finance management application. It allows users to register accounts, credit cards, transactions, budgets, and more. The goal is to provide a clear overview of the available balance and project the expected balance at the end of each cycle - even if not all funds have been spent yet.

## Functionalities

- Users 
- Accounts
- Credit cards
- Categories
- Cycles
- Transactions
- Budgets
